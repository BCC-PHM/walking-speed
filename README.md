# BSol Walking Speed Estimates

Estimating the percentage of residents in each LSOA unable to walk at the 1.2m/s speed required to cross the road at a pedestrian crossing. Age and sex walking impairment percentages based on Asher et. al: https://doi.org/10.1093/ageing/afs076.
